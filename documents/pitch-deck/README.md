# Pitch Deck

Place pitch deck files here.

## Recommended Formats

- `.pptx` — PowerPoint (editable)
- `.key` — Keynote (editable, Mac)
- `.pdf` — Export for sharing (non-editable)

## File Naming Convention

```
sistemahub-pitch-deck-v1.pptx
sistemahub-pitch-deck-v1.pdf
sistemahub-pitch-deck-v2.pptx   ← bump version when making major changes
```

## Suggested Slide Structure

1. Cover — Company name, tagline, date
2. Problem — What problem are we solving?
3. Solution — What does SistemaHub do?
4. Products — TaoLink HRIS, Landlord HQ
5. Market — Who are the customers?
6. Business Model — How do we make money?
7. Traction — Any clients, users, or milestones?
8. Team — Who's behind SistemaHub?
9. Ask — What are we looking for?
10. Contact — hello@sistemahub.com
