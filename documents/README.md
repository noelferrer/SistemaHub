# Documents

All client-facing and internal business documents live here.

## Structure

```
documents/
├── company-profile.md          ← Who we are — used in proposals and partnerships
├── pitch-deck/                 ← Pitch deck files (slides, exports)
└── templates/
    ├── proposal-template.md    ← Standard proposal template
    └── invoice-template.md     ← Standard invoice template
```

## When to Use Each

| Document | Use when |
|----------|----------|
| `company-profile.md` | Attaching to proposals, partnership discussions, introductory emails |
| `pitch-deck/` | Investor meetings, client presentations |
| `templates/proposal-template.md` | Sending a project proposal to a client |
| `templates/invoice-template.md` | Billing a client after project completion |
