# Logo Files

Place all logo exports here.

## Required Files

| File | Description |
|------|-------------|
| `logo-primary.svg` | Full logo, horizontal layout |
| `logo-icon.svg` | Icon/mark only (for favicons, app icons) |
| `logo-dark.svg` | For use on dark/colored backgrounds |
| `logo-light.svg` | For use on light/white backgrounds |
| `logo-primary.png` | PNG at 1x resolution |
| `logo-primary@2x.png` | PNG at 2x (retina) resolution |

## Usage Rules

- Never stretch or distort the logo
- Maintain clear space around the logo equal to the height of the icon
- Do not place the primary logo on busy backgrounds — use the appropriate dark/light version
- Minimum size: 120px wide for digital use
