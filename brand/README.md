# Brand Assets

This folder contains all SistemaHub visual identity files.

## Structure

```
brand/
├── logo/               ← All logo files go here (see logo/README.md)
├── brand-guidelines.md ← Colors, typography, tone of voice, usage rules
└── README.md           ← This file
```

## What Goes in logo/

- `logo-primary.svg` — Full logo (horizontal)
- `logo-icon.svg` — Icon/symbol only
- `logo-dark.svg` — Version for dark backgrounds
- `logo-light.svg` — Version for light backgrounds
- `logo-primary.png` — PNG export (1x)
- `logo-primary@2x.png` — PNG export (2x / retina)
